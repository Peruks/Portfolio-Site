# Portfolio Website

A modern, responsive personal portfolio website designed for an AI & Machine Learning enthusiast. This project features a futuristic "Data Matrix" theme with interactive animations and a clean, professional layout.

## Features

-   **Data Matrix Theme**: A dark, futuristic aesthetic with neon cyan and green accents, inspired by data science and AI.
-   **Interactive Hero Section**: Features a custom "Digital Data Matrix" background animation (Matrix rain + Neural Network nodes) built with HTML5 Canvas.
-   **Responsive Design**: Fully responsive layout that adapts seamlessly to desktops, tablets, and mobile devices.
-   **Glassmorphism UI**: Modern glass-effect cards for About, Skills, and Project sections.
-   **Clean Navigation**: Sticky navbar with smooth scrolling to sections.

## Tech Stack

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling with CSS variables, Flexbox, and Grid. No external frameworks used for layout.
-   **JavaScript**: Vanilla JS for the canvas animation and interactivity.
-   **Icons**: [Lucide Icons](https://lucide.dev/) for consistent and beautiful iconography.
-   **Fonts**: [Inter](https://fonts.google.com/specimen/Inter) from Google Fonts.

## Getting Started

To view this project locally, you can use any static file server. Since Python is commonly installed, you can use its built-in HTTP server:

1.  Clone the repository:
    ```bash
    git clone https://github.com/Peruks/Portfolio-Site.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Portfolio-Site
    ```
3.  Start the server:
    ```bash
    python -m http.server 8000
    ```
4.  Open your browser and visit `http://localhost:8000`.
